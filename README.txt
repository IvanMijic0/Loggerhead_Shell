Welcome to the Loggerhead Shell repository! This repository contains a basic shell implemented through a C script that can handle piping and redirection, along with basic terminal commands.

# Files

- basic_program.c: C program demonstrating how to create a new process using fork() and execute a different program using execvp().
- intermediate_program: C program creating a child process using fork() and executing the "ls -l" command using execvpe().
- loggerhead_shell.c: Shell implementation script.
- Questions.txt: Answered questions for task 1.5.
